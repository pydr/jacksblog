# Jack's Blog
### 项目简介 Abstract

一个基于Django2.0的个人博客系统

A blog website base on Django2.0



### 技术栈 Stack

* **后台 Backend**
  * Python 3.5
  * Django 2.0
  * Django REST Framework
* **数据库 Database**
  * Mysql
  * Redis
* **前端 Frontend**
  * Jquery
  * Vue.js
